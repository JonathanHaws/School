Git

Git is powerful, efficent, and crucial technology that stores multiple versions of things by only recording changes. Another key feature is branching / merging. Combine this with cloud storage and you have a must know feature industry is completely dependant on to collaborate

  • git revert: Changes clone to older hash version
  • git clone repository: Creates copy of repo on local computer
  • git remote -v: Verifies connection & ability to push & fetch
  • git status: returns current Branch clone represents
  • git checkout -b newbranch name: Creates new branch
  • git checkout -f branchname: Moves to branch and ignores all changes and erases them
  • git add -A: Stages all files to be commited
  • git commit -m "Message to be in commit": Soldifies changes
  • git pull origin branchname: Makes sure that branch is up to date
  • git push origin branchname: Finally uplods to cloud repository
